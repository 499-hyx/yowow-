# 提示词路由表 · 三层模型

```text
层1 抓取        每日公共抓取  +  每个赛道定向抓取   → 中立化 → 热点池
                  ↓
层2 判断        判断能不能宣发（strong_pick / maybe / skip）
                  ↓（只对能宣发的）
层3 赛道分析    每个赛道专属分析提示词 → 剖析成可发布内容
                  ↓
                ingest.py 安装 → data/today → 前端
```

|---|---|---|---|---|
| 1 | 公共抓取（大流量/在发生的事，无行业字眼） | `prompts/hotspot-broad.md` | 人/agent 手工填 `{date}` | 博士/老板 |

## 每天怎么跑（命令序列）

```bash
D=2026-06-23   # 改成当天

# 层1 抓取：手工渲染 → 写热点池
#   公共池  : 用 prompts/hotspot-broad.md 作答 → 每条过 prompts/neutralize.md → data/hotspots/$D.json（scope:"broad"）
#   赛道池  : 用 prompts/tracks/education-yowow/search.md 作答 → 过中立化 → data/hotspots/tracks/education-yowow/$D.json
python3 scripts/status.py --date $D            # 自检：两个池子在不在

# 层2 判断
python3 scripts/make-prompt.py acct-xiaozhu-edu-xhs --date $D --step match
#   逐条答 data/runs/$D/acct-xiaozhu-edu-xhs/prompts/match-*.txt → 写 _inbox/match-<id>.json（tier）

# 层3 赛道分析（只对非 skip）：出稿时对照 prompts/tracks/education-yowow/analysis.md 组织 body
python3 scripts/make-prompt.py acct-xiaozhu-edu-xhs --date $D --step generate
#   写 _inbox/generate-<id>.json（≥3 桥 5 步 + content 三件套 + external_terms_check:true）

# 安装（唯一能写 data/today 的入口）
python3 scripts/ingest.py acct-xiaozhu-edu-xhs data/runs/$D/acct-xiaozhu-edu-xhs/_inbox --date $D
python3 scripts/status.py --date $D            # 自检：期望 Missing:(none)
```
