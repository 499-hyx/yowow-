# 任务：抓取过去 7 天具有范式转移意义的全网热点

今天是 {date}。请检索过去 7 天内发生的、具有“范式转移”意义的全球及中国热点事件，优先覆盖 AI、教育、实业、民生、重大公共事件、家庭教育观念变化、普通人生活结构变化。

这不是内容创作任务。你只负责找出值得进入公共热点池的原料，并输出系统可直接保存的 JSON 数组。

<!-- <<< 可编辑区 开始 ================================ 博士 / 老板在此调筛选口径 -->

## 搜索范围

1. **顶级科技 / 商业领袖的深度言论**
   - OpenAI、Anthropic、Google DeepMind、微软、Nvidia、Meta 等掌门人及首席科学家的播客、采访、演讲、长文。
   - 中国头部 AI 公司创始人、顶级投资人、教育家、创业者关于未来教育和人才培养的公开言论。

2. **突破性的科学 / 教育发现**
   - 脑科学、认知科学、儿童认知发展、学习机制研究。
   - AI 在数学、物理、科学研究中的突破。
   - 学习方式、考试制度、素质教育、职业教育的新变化。

3. **全球公共政策的底层逻辑变动**
   - 与教育、AI、实业、民生、就业、技能重构直接相关的政策变化。
   - 人口结构、未来工作、终身学习、大学学位价值变化。

4. **平台高互动话题**
   - 中文平台：小红书、抖音、微博、知乎、B站、得到、播客。
   - 国际平台：X / Twitter、LinkedIn、YouTube、Medium、Reddit。
   - 影视剧、纪录片、公开演讲中与教育、家庭、AI、未来工作相关的热点。

## 入池硬门槛

一条热点能入池，必须至少命中下面一个信号：

1. **逆转信号**：某个长期共识被顶级玩家用行动或言论反转。
2. **天花板信号**：重新定义了 AI、教育、组织或个人能力的上限 / 下限。
3. **重构信号**：个人注意力、学习方式、组织流程、产业链发生实质性重构。
4. **稀缺性反转信号**：AI 让某种原本廉价的能力变稀缺，或让原本稀缺的能力变廉价。
5. **认知差信号**：大众叙事和精英行动之间出现明显反差。

只是“热度高”但没有上述意义的，不入池。

## 排除项

- 排除纯娱乐八卦、明星私生活，除非它直接引发家庭 / 教育 / AI / 未来工作层面的结构性讨论。
- 排除单纯产品参数介绍。
- 排除带货直播琐事。
- 排除表面排名新闻。
- 排除与教育、未来、AI、实业、民生无关的泛热点。
- 排除监管立法、涉诉、天灾人祸蹭热点、失业焦虑贩卖、细节不足以支撑深度剖析的条目。

## 分析口径

对每条热点，在内部先回答：

1. 这件事表面上是什么？
2. 它背后“谁在干什么”？
3. 它暴露了什么客观现实？
4. 它说明什么能力、资源、关系或组织方式正在变贵 / 变便宜？
5. 普通家庭、普通创业者或普通内容运营者为什么应该关注它？

但最终不要输出长篇分析，只把分析压缩进 JSON 字段。

## 输出数量

- 默认输出 10-20 条。
- 找不到足够可靠的范式转移级热点时，宁可少输出，不要凑数。
- 高风险但重要的热点可以保留，但必须标注 `risk_level` 和 `risk_notes`。
- 信息源不清、无法核实、纯情绪、纯八卦、纯产品参数的热点不要入主数组。

## 护栏

- `hotspot_id` 必须唯一，建议按 `hs-YYYYMMDD-ultimate-001` 递增。
- `source_url` 没有就填 `null`，不要编造链接。
- `source_notes` 必须有，至少说明平台、媒体、访谈、论文或榜单来源。
- `verification_status` 不清楚时用 `needs_verification`，不要写成已确认。
- 不要输出内部成稿术语，不要给发布建议，不要写脚本，不要写标题模板。
