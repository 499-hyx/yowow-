# 任务：为「科学护肤」赛道定向召回热点候选

今天是 {date}。请搜索过去 72 小时内，能自然连接到护肤用户真实问题、科学护肤认知和产品选择的真实热点。

## 搜索方向

1. 护肤圈热门事件：品牌、产品、达人、平台内容引发的真实讨论。
2. 护肤产品翻车或爆火：防晒、修护、抗老、美白、祛痘、清洁、面膜等品类的用户反馈和争议。
3. 皮肤健康热点：敏感、泛红、长痘、毛孔、暗沉、屏障受损、换季不适等大众可理解问题。
4. 防晒与高温暴晒：夏季通勤、旅行、户外运动、晒后护理、补涂防晒等讨论。
5. 成分党热门讨论：A 醇、视黄醇、烟酰胺、VC、传明酸、神经酰胺、玻尿酸、壬二酸、水杨酸、果酸、早 C 晚 A 等。
6. 护肤误区与踩坑案例：过度清洁、盲目刷酸、叠加太多产品、跟风买不适合自己的产品。
7. 医美和轻医美讨论：只收大众公开讨论和消费决策，不做医疗诊断，不收违规宣传。
8. 生活方式导致的皮肤问题：熬夜、压力、饮食、运动出汗、口罩闷脸、空调房干燥等。

## 召回口径（在此编辑）

1. **只收自然能讲护肤认知的热点。** 普通用户看完会自然想到皮肤状态、护肤选择或产品踩坑，才算召回；需要绕很远才接到护肤的不收。
2. **优先真实用户问题，不优先纯带货热闹。** 有 UGC 争议、使用反馈、成分讨论、场景痛点的优先；只有品牌露出、明星同款、低价促销的不优先。
3. **时间窗以过去 72 小时为主。** 常青护肤知识不算热点；除非最近被平台、媒体、达人或用户重新点燃。
4. **必须可核验。** 尽量给媒体、平台、话题、帖子或检索线索；记不清出处、疑似编造、不知道谁在讨论的删掉。
5. **每条方向召回 0-2 条即可。** 召回不到就空着，不为凑数降低标准。
6. **同一事件只出一次。** 如果命中多个方向，在 `hit_directions` 里全部写出。

## 不要收

- 需要承诺「根治、永久、100%有效、一夜变白、七天见效、零副作用、绝对安全」才能讲下去的内容。
- 处方药、诊断治疗、严重皮肤病个案、违法医疗广告、违规医美宣传。
- 没有事实细节的营销通稿、品牌自嗨、纯明星八卦、纯娱乐流量。
- 只能制造容貌焦虑，不能帮助用户建立护肤判断的内容。
- 需要攻击竞品或贬低用户才能成立的话题。

## 输出格式（严格输出下面这个 JSON 数组，不要多余文字）

每条都要直接符合系统热点池格式，可以原样粘回运营台保存。不要先输出散文、表格、解释或二次整理说明。

## 护栏

- 召回为空就输出 `[]`，绝不编造事件、声量、来源或用户反馈。
- `why_natural` 写不顺的条目，删掉。
- 本步只负责召回和格式化入池，不做最终赛道归属判断、不写内容方案。
- `hotspot_id` 必须唯一；同一天建议用 `hs-YYYYMMDD-skincare-001` 递增。
- 输出必须是 JSON 数组，不能输出 Markdown 表格。
